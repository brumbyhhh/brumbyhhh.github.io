Button 按钮的实现

  • 实现思路很简单，就是将 html 的 Button 加上一些 css 样式，并让使用者可以传入 Button 的类型，例如 ghost 类型。

    • Button.tsx 文件

      • 定义 Button 的一些属性
      interface ButtonProps {
  onClick?: () => void;
  style?: React.CSSProperties;
  variant?: "default" | "dashed" | "outline" | "ghost";
  children: React.ReactNode;
}
      • 渲染 Button 组件
      const Button: React.FC<ButtonProps> = ({
  onClick,
  style,
  variant = "default",
  children,
}) => {
  return (
    <button className={`button ${variant}`} onClick={onClick} style={style}>
      {children}
    </button>
  );
};

export default Button;

    • 引入 CSS 样式

      • Button.css 文件
      /* 默认样式 */
.button {
  background-color: #0967cc;
  color: white;
  border: none;
  padding: 10px 20px;
  cursor: pointer;
}

/* 虚线框样式 */
.button.dashed {
  border: 1px dashed #0967cc;
  background-color: transparent;
  color: #007bff;
}

/* 无框样式 */
.button.outline {
  border: 1px solid #0967cc;
  background-color: transparent;
  color: #007bff;
}

/* 幽灵样式 */
.button.ghost {
  background-color: transparent;
  color: #007bff;
}

/* 鼠标悬停时的样式 */
.button:hover {
  background-color: #0056b3;
  border-color: #0056b3;
}

/* 点击时的样式 */
.button:active {
  background-color: #007bff;
  border-color: #003d80;
}

/* 获取焦点时的样式 */
.button:focus {
  outline: none;
  box-shadow: 0 0 4px rgba(0, 123, 255, 0.5);
}

Button 的使用

  • 我们的 Button 已经封装好了，在项目中可以使用我们自己封装的 Button 组件
import { Button } from "../../wanwu-ui/src/index";

function App() {
  return (
    <>
      <div>
        <Button variant="outline" onClick={() => alert("点击按钮")}></Button>
      </div>
    </>
  );
}

export default App;
